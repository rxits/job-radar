# job-radar v2 Phase 2 — UX Rebuild Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Replace the overwhelming kanban-wall with a guided product: first-run setup wizard, a ranked "Today" feed (Apply / Skip / Save per job), a pipeline view of only-acted-on jobs, and one Refresh button that chains scrape→match.

**Architecture:** Feed/seen/starred mechanics ride on Phase 1's columns (`seen_at`, `starred`). `listJobs` grows three filter flags + a richer ORDER BY. One new API route (`/api/refresh`) chains the existing orchestrators. Board component moves to `/pipeline` filtered to actioned jobs; `/` becomes the feed; `/setup` is a 3-step client wizard guarded by a server-side profile check.

**Tech Stack:** unchanged. Spec: `docs/superpowers/specs/2026-06-11-job-radar-v2-design.md` (Phase 2).

**Design deviation (deliberate):** the feed shows `eligible + unknown` (not eligible-only as the spec sketch says) — before the user's first Gemini pass most jobs are unknown and an empty feed would be a broken first impression. Unknowns rank below scored jobs naturally (score sort, NULLS LAST) and wear the `?` badge.

```ts
import { NextResponse } from "next/server";
import { db } from "@/lib/server-db";
import { runScrape } from "@/lib/scrape";
import { matchNew, geminiClient } from "@/lib/match";

export const maxDuration = 300;

export async function POST() {
  const scrape = await runScrape(db());
  let match: { scored: number; failedBatches: number; failedJobs: number } | { error: string };
  try {
    match = await matchNew(db(), geminiClient());
  } catch (e) {
    match = { error: e instanceof Error ? e.message : String(e) };
  }
  return NextResponse.json({ scrape, match });
}
```

- [ ] **jobs PATCH** — replace body handling:

```ts
export async function PATCH(req: Request) {
  let body: any;
  try { body = await req.json(); } catch { return NextResponse.json({ error: "invalid json" }, { status: 400 }); }
  const { id, status, seen, starred } = body ?? {};
  if (!id) return NextResponse.json({ error: "id required" }, { status: 400 });
  if (status !== undefined) {
    if (!STATUSES.includes(status)) return NextResponse.json({ error: "invalid status" }, { status: 400 });
    db().setStatus(id, status as Status);
  }
  if (seen === true) db().markSeen(id);
  if (starred !== undefined) db().setStarred(id, !!starred);
  if (status === undefined && seen === undefined && starred === undefined)
    return NextResponse.json({ error: "nothing to update" }, { status: 400 });
  return NextResponse.json({ ok: true });
}
```
